## Key Decisions

| Decision | Rationale |
|----------|-----------|
| aiosqlite direct (no ORM) | Cache layer, not primary storage — simpler |
| Public repo | Free-tier GitHub needs public for branch protection rulesets |
| Conventional commits | feat:, fix:, chore: — trunk-based with PR flow |
| Regex parsing over markdown AST | No new deps, structured md has consistent formatting |
| Flat cache table + JSON columns | Simpler queries for blockers/decisions lists |
| Per-plan feature branches | Squash merge to main, clean branch per plan |
| Per-change DB connection in watcher | Simpler lifecycle, no stale connections |
